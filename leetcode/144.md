# 144 二叉树的前序遍历

迭代

```java
class Solution {
    public List<Integer> preorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        LinkedList<TreeNode> stack = new LinkedList<>();
        TreeNode cur = root;
        while (cur != null || !stack.isEmpty()) {
            while (cur != null) {
                stack.addLast(cur);
                res.add(cur.val);
                cur = cur.left;
            }
            cur = stack.removeLast();
            cur = cur.right;
        }
        return res;w
    }
}
```
