# 10-2 青蛙跳台阶问题

动态规划

```java
class Solution {
    public int fib(int n) {
        int[] arr = new int[n + 1];
        if (n == 0) return 0;
        arr[0] = 0;
        arr[1] = 1;
        for(int i = 2; i <= n; i++) {
            arr[i] = (arr[i - 1] + arr[i - 2]) % 1000000007;
        }
        return arr[n];
    }
}
```
