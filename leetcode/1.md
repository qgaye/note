# 1 两数之和

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        HashMap<Integer, Integer> map = new HashMap<Integer, Integer>();
        for (int i = 0; i < nums.length; i++) {
            int j = target - nums[i];
            if (map.containsKey(j)) {
                return new int[] {i, map.get(j)};
            }
            map.put(nums[i], i);
        }
        return new int[] {-1, -1};
    }
}
```
