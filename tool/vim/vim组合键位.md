# vim组合键位

| 键位 | 功能 |
|:-:|:-:|
| `Ctrl + A` | Normal模式下对数字加一 |
| `Ctrl + X` | Normal模式下对数字减一 |
| `Ctrl + C` | Insert模式下等同于`ESC` |
| `Ctrl + [` | Insert模式下等同于`ESC` |
| `Ctrl + O` | 上次光标所在位置(不仅仅是当前已打开的文件) |
| `Ctrl + E` | 向下滚动一行 |
| `Ctrl + Y` | 向上滚动一行 |
| `Ctrl + D` | 向下滚动半屏 |
| `Ctrl + U` | 向上滚动半屏 |
| `Ctrl + F` | 向下滚动一屏 |
| `Ctrl + B` | 向上滚动一屏 |
| `Ctrl + V` | 进入`Visual Block`模式 |
| `Shift + V` | 进入`Visual Line`模式 |
